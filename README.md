# Community Engagement Platform

A modern full-stack web application that empowers citizens to report community issues and enables government officials to manage and resolve them efficiently.

The Community Engagement Platform bridges the gap between citizens and government officials by providing a streamlined system for reporting, tracking, and resolving community issues. Citizens can easily report problems with visual documentation, while administrators can efficiently manage and update issue statuses through a dedicated admin panel.

## ‚ú® Features

### For Citizens
- **User Authentication** - Secure registration and login system
- **Issue Reporting** - Report community problems with photo uploads
- **Real-time Tracking** - Monitor issue status and resolution progress
- **Public Dashboard** - Browse all community issues with advanced filtering
- **Search Functionality** - Quick search by title, location, or category
- **Export Data** - Download filtered issue data as CSV

### For Government Officials
- **Admin Panel** - Dedicated interface for issue management
- **Status Management** - Update issue status through workflow stages
- **Analytics Dashboard** - View statistics and insights at a glance
- **Priority System** - Categorize issues by urgency level
- **Photo Evidence** - Review visual documentation of reported issues

### Advanced Capabilities
- **Multi-level Filtering** - Filter by status, category, and priority simultaneously
- **Real-time Analytics** - Live statistics showing issue distribution
- **Category System** - Organized classification (Infrastructure, Sanitation, Safety, etc.)
- **Priority Levels** - Critical, High, Medium, and Low priority management
- **Responsive Design** - Professional, elegant interface for all devices
